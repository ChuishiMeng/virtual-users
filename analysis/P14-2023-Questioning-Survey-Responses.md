## 📋 基本信息

**研究主题**: 批判性检验用调查问卷研究 LLM 的方法论

**核心问题**: 
- LLM 的调查回答能否反映真实人类群体？
- 调查衍生的对齐指标是否可靠？
- LLM 调查回答有什么系统性偏差？

**核心结论**: **LLM 调查回答不可靠** ⚠️⚠️

---

## 📊 关键发现

### 1. 顺序和标签偏见 ⚠️

**实验设置**：
- 数据集：美国社区调查（ACS），25个多选题
- 模型：43个不同规模的 LLM
- 方法：记录答案概率分布

**发现**：
```
LLM 倾向于选择标签为 "A" 的答案！
```

**证据**：
- 强烈的顺序偏见（order bias）
- 强烈的标签偏见（labeling bias）
- 模型回答受答案排序和标签影响

### 2. 调整后趋向均匀随机 ⚠️⚠️

**解决方案**：随机化答案排序

**结果**：
```
调整后，所有模型趋向于均匀随机回答
与模型规模、预训练数据无关
```

**对比**：
| 条件 | 回答分布 |
|------|---------|
| **原始** | 受顺序/标签偏见影响 |
| **随机化后** | 趋向均匀随机（高熵）|
| **人类群体** | 自然熵变化（低-中）|

**关键图表**（图2）：
- 模型熵随规模对数线性增长
- 人类群体熵自然变化
- 模型熵 > 任何人类群体

### 3. 对齐指标的简单解释 ⚠️⚠️⚠️

**先前研究的猜测**：
> "模型与某些人群更相似，是因为这些人更好地被训练数据代表"

**本文的简单解释**：
> "模型与某些人群更相似，只是因为这些人的回答熵更接近均匀分布"

**证据**：
- 对齐指标 = 与均匀分布的接近度
- 与训练数据、训练过程无关
- **可完全由熵解释**

## 🔍 详细分析

### 1. 熵的增长模式

**实验**：
- 43个模型，不同规模
- 25个问题，独立提问
- 记录答案分布熵

**结果**（图2）：
```
熵 ∝ log(模型规模)

小模型（<1B）：熵较低
中模型（1B-70B）：熵中等
大模型（>70B）：熵较高，接近均匀
```

**人类对比**：
```
美国人口：熵 = 0.3-0.8（自然变化）
加利福尼亚：熵 = 0.35-0.85
德克萨斯：熵 = 0.3-0.8

LLM（>70B）：熵 ≈ 0.9-1.0（接近均匀）
```

**结论**：
- LLM 熵 > 任何人类群体
- LLM 趋向均匀随机

### 2. 指令微调的影响

**实验**：
- 对比基础模型 vs 指令微调模型
- 评估回答分布与人类群体的差异

**结果**：
```
指令微调后：
- 小模型：熵增加，差异不变
- 大模型（>70B）：熵略降，差异略减

趋势：指令微调不能解决均匀化问题
```

### 3. 跨调查验证

**数据集**：
- ACS（美国社区调查）
- 其他调查（先验研究使用的）

**发现**：
```
无论使用什么调查：
- 模型趋向均匀回答
- 对齐指标可由熵解释
- 与训练数据无关
```

## 🚀 可直接应用的技术

### 1. 随机化答案顺序

```python
import random

def randomized_survey_prompt(question, options):
    """随机化答案顺序"""
    # 随机打乱选项
    shuffled = list(enumerate(options))
    random.shuffle(shuffled)
    
    # 重新标签
    labels = ['A', 'B', 'C', 'D']
    prompt = f"Question: {question}\n\n"
    for i, (orig_idx, option) in enumerate(shuffled):
        prompt += f"{labels[i]}. {option}\n"
    
    prompt += "\nAnswer:"
    
    return prompt, shuffled
```

### 2. 熵计算和监控

```python
import numpy as np

def calculate_entropy(prob_distribution):
    """计算回答分布熵"""
    # 移除零值
    probs = np.array(prob_distribution)
    probs = probs[probs > 0]
    
    # 计算熵
    entropy = -np.sum(probs * np.log2(probs))
    
    return entropy

def check_uniform_bias(entropy, max_entropy):
    """检查均匀化偏见"""
    ratio = entropy / max_entropy
    
    if ratio > 0.9:
        print(f"⚠️ 警告：回答趋向均匀（熵比 = {ratio:.2f}）")
        return True
    
    return False
```

### 3. 对齐指标的正确解释

```python
def alignment_analysis(model_dist, human_dist, uniform_dist):
    """分析对齐指标的真实原因"""
    # 计算与人类群体的距离
    human_kl = kl_divergence(model_dist, human_dist)
    
    # 计算与均匀分布的距离
    uniform_kl = kl_divergence(model_dist, uniform_dist)
    
    # 判断对齐原因
    if uniform_kl < human_kl:
        print("⚠️ 对齐指标由均匀分布解释，不是训练数据代表谁")
        return "uniform"
    else:
        print("✅ 对齐指标可能反映真实训练数据")
        return "data"
```
