## 📋 基本信息

**研究主题**: 从不确定性量化视角确定LLM相当于多少人类受访者

**核心问题**: 
- 用LLM模拟调查时应该生成多少样本？
- 如何量化LLM的模拟保真度？
- LLM相当于多少人类受访者？

**核心贡献**:
1. **通用框架**: 将LLM模拟响应转换为可靠置信区间
2. **自适应样本量选择**: 数据驱动选择模拟样本数量
3. **三大洞察**: 实用指南、对齐度量、有效人类样本量

**关键词**: Synthetic data, LLMs, Uncertainty quantification, Simulation

---

## 🎯 研究动机

### LLM模拟的困境

**核心矛盾**：

**生成太多样本**：
- 过度拟合合成分布
- 与真实人群偏差
- 置信区间过窄，覆盖率差

**生成太少样本**：
- 置信区间过宽
- 信息量不足
- 不实用

**关键洞察**：
> "最优样本量取决于真实和合成响应分布的差异，这在实践中是未知的。"

### 研究目标

**核心问题**：如何用LLM生成数据构建人类响应总体参数的可靠置信区间？

**三大贡献**：

1. **实用模拟指南**：指定应生成的合成样本数量
2. **直观对齐度量**：置信区间宽度反映LLM与人群的偏差
3. **有效人类样本量**：回答"LLM相当于多少人类样本？"

## 💡 核心方法

### 1. 问题设定

**场景**：
- 目标：估计人类响应的总体参数μ（如均值）
- 传统方法：测试n个真实人，得到y₁,...,yₙ
- LLM方法：用LLM生成k个合成响应

**置信区间**（传统）：
```
[ȳ - z_{α/2} · s/√n, ȳ + z_{α/2} · s/√n]
```

**置信区间**（LLM合成）：
```
[ȳ_syn^k - z_{α/2} · s_syn^k · √(C/k), ȳ_syn^k + z_{α/2} · s_syn^k · √(C/k)]
```

其中C > 1是膨胀因子（补偿分布偏差）

### 2. 样本量选择方法

**校准集**：
- m个类似调查问题
- 每个问题有真实人类数据Dⱼ
- 每个问题用LLM生成K个合成响应

**误覆盖率度量**：
```
G(k) = (1/m) Σ_{j=1}^m 1{ȳⱼ ∉ I_syn^j(k)}
```

**选择准则**：
```
k̂ = max {0 ≤ k ≤ K : G(i) ≤ α/2 ∀i ≤ k}
```

**直觉**：
- 选择最大的k，使得所有更小的样本量的误覆盖率都≤α/2
- α/2阈值补偿用ȳⱼ作为μⱼ代理的额外采样误差

### 3. 理论保证

**覆盖有效性**（Theorem 2.1）：
```
P(μ ∈ I_syn(k̂)) ≥ 1 - (1-2η)^(-1) [α + O(√(log(1/δ)/m))]
```

其中η是校准集和测试集的分布偏差

**尖峰性**（Sharpness）：
- k̂接近oracle样本量k*（真实最优）
- 相对误差|k̂ - k*| / k* < 0.25（95%情况下）

### 4. "机械土耳其人"模型

**模型设定**：
- LLM = 由κ个真实人类组成的"机械土耳其人"
- 每次LLM生成 = 从这κ人中随机抽样一个响应

**关键结果**（Theorem 4.1）：
```
在"机械土耳其人"模型下，k̂ ≈ κ/C
```

**解释**：
- k̂反映了LLM能有效代表的人群规模
- κ̂ = k̂/C是"隐藏人群大小"的估计

## 🔍 关键洞察

### 1. 对虚拟用户研究的启示 ⭐⭐⭐

**核心价值**：
1. ✅ **量化保真度方法**：κ̂ = k̂/C可量化LLM相当于多少人类
2. ✅ **跨领域异质性**：揭示不同领域的保真度差异
3. ⚠️ **保真度有限**：最高58人（GPT-4o），远低于真实调查规模

**与P11+P12+P14+P18的呼应**：

| 论文 | 核心警告 | P19验证 |
|------|---------|---------|
| P11 Lost in Simulation | 人口偏见 | ✅ 不同领域κ̂差异大 |

**综合结论**：

### 2. 技术要点

**样本量选择算法**：
```python
def select_simulation_sample_size(calibration_set, alpha=0.05):
    """选择模拟样本量"""
    K = len(calibration_set[0]['synthetic_responses'])
    
    for k in range(K, 0, -1):
        # 计算误覆盖率
        G_k = 0
        for question in calibration_set:
            CI = construct_CI(question['synthetic_responses'][:k])
            if question['human_mean'] not in CI:
                G_k += 1
        G_k /= len(calibration_set)
        
        # 检查所有i≤k
        if all(G_i <= alpha/2 for i in range(1, k+1)):
            return k
    
    return 0
```

**置信区间构建**：
```python
def construct_CI(synthetic_responses, alpha=0.05, C=2.0):
    """构建膨胀置信区间"""
    k = len(synthetic_responses)
    mean = np.mean(synthetic_responses)
    std = np.std(synthetic_responses)
    
    z = stats.norm.ppf(1 - alpha/2)
    half_width = z * std * np.sqrt(C / k)
    
    return [mean - half_width, mean + half_width]
```

### 3. 局限性

**方法假设**：
1. ⚠️ **需要校准集**：需要m个类似问题的真实数据
2. ⚠️ **分布相似假设**：假设测试集与校准集分布相似
3. ⚠️ **二元响应**：主要针对二元响应（是/否）

**实践限制**：
1. ⚠️ **计算成本**：每个校准问题需要生成K个合成响应
2. ⚠️ **超参数C**：膨胀因子C需要调优
3. ⚠️ **领域依赖**：不同领域需要不同校准集
