## 一、已下载可用数据集

### 1.1 Airline 乘客满意度（主数据集 ✅）

| 属性 | 值 |
|------|-----|
| **文件** | airline_train.csv (14.7MB), airline_test.csv (3.0MB) |
| **样本量** | 105,270 训练 + 25,976 测试 |
| **评分问题** | 12 个服务评分（1-5分） |
| **人口统计** | Gender, Age, Customer Type, Type of Travel, Class |
| **目标变量** | satisfaction (满意/不满意) |

**跨问题一致性验证（已分析）**：
| 约束条件 | 满意概率 | 约束强度 |
|---------|---------|---------|
| 评分均值 > 2.5 | 49.3% | 弱约束 |
| 评分均值 > 3.0 | 59.0% | 中约束 |
| 评分均值 > 3.5 | **72.2%** | ✅ 可用约束 |
| 评分均值 > 4.0 | **88.1%** | ✅ 强约束 |

**可用 CQCB 约束示例**：
```
C1: 总体满意度约束
    IF 评分均值 > 3.5 THEN satisfied概率 > 70%
    
C2: 关键服务约束
    IF Inflight wifi service > 4 AND Online boarding > 4 THEN satisfied概率 > 85%
    
C3: 客户类型一致性
    Loyal Customer 的评分方差应小于 Disloyal Customer（态度更稳定）
```

## 五、CQCB 约束构建计划

**从 Airline 数据提取约束**：

| 约束类型 | 数量 | 提取方法 |
|---------|------|---------|
| 评分-满意度关联 | ~10 条 | 统计关联分析（评分均值→满意概率） |
| 服务组合约束 | ~15 条 | 多服务评分组合→满意度预测 |
| 客户类型一致性 | ~5 条 | Loyal/Disloyal 评分模式差异 |
| 旅行类型差异 | ~5 条 | Business/Personal 评分权重差异 |

**验证方法**：
- 在真实数据上验证约束满足率 > 70%
- 过滤无效约束（满足率 < 50% 或满足率 > 95% 过于平凡）
