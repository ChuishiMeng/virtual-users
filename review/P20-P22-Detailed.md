## P20 - Credibility Evaluating LLMs（12页）

### 📋 基本信息

**论文ID**: 2026
**研究主题**: 评估LLM生成内容的可信度
**页数**: 12页
**评分**: **7/10 (P1)** ⭐⭐

### 💡 核心内容

**研究问题**：
- 如何评估LLM生成内容的可信度？
- LLM在事实准确性、逻辑一致性上的表现如何？

**核心贡献**：
1. **可信度评估框架**：多维度评估（准确性、一致性、完整性）
2. **评估指标**：设计量化指标
3. **基准测试**：在多个任务上评估LLM

### 🔍 关键发现

**LLM可信度问题**：
- ⚠️ **幻觉问题**：生成看似合理但不准确的内容
- ⚠️ **不一致性**：相似问题给出矛盾答案
- ⚠️ **偏见放大**：训练数据偏见被放大

**评估维度**：
1. **事实准确性**：与已知事实对比
2. **逻辑一致性**：内部逻辑是否自洽
3. **来源可追溯性**：是否提供来源

### 🎯 对虚拟用户研究的启示

**相关性**：中等（7/10 P1）

**核心价值**：
1. ✅ **评估框架**：可用于评估虚拟用户回答质量
2. ✅ **一致性检查**：验证虚拟用户响应一致性
3. ⚠️ **可信度问题**：LLM生成内容需验证

**实践建议**：
- 用P20的可信度评估框架检查虚拟用户响应
- 实现一致性检查（相似问题是否给出一致答案）
- 建立来源追溯机制（虚拟用户回答基于什么信息）

- 现有评估方法的优缺点是什么？

**核心贡献**：
1. **评估方法分类**：任务导向、能力导向、行为导向
2. **评估指标综述**：准确率、F1、一致性、鲁棒性等

### 🔍 关键发现

| 类型 | 焦点 | 示例 |
|------|------|------|
| **任务导向** | 特定任务性能 | 问答、对话、代码生成 |
| **能力导向** | 通用能力 | 推理、规划、工具使用 |
| **行为导向** | 行为特征 | 一致性、安全性、偏见 |

**评估挑战**：
1. ⚠️ **主观性**：某些任务难以量化
2. ⚠️ **成本高**：人类评估昂贵

**最佳实践**：
1. ✅ **多维度评估**：结合多种指标
2. ✅ **自动化+人工**：混合评估方法
3. ✅ **持续评估**：模型更新后重新评估

## P22 - SimUSER（18页）

### 📋 基本信息

**论文ID**: 2025
**研究主题**: 用户行为模拟系统
**页数**: 18页
**评分**: **8/10 (P1)** ⭐⭐⭐

### 💡 核心内容

**研究问题**：
- 如何用LLM模拟用户行为？
- 在推荐系统场景中效果如何？

**核心贡献**：
1. **SimUSER框架**：用户行为模拟系统
2. **三模块设计**：Profile + Memory + Action
3. **推荐场景验证**：在多个推荐数据集上测试

### 🔍 关键发现

**三模块架构**：

```
┌─────────────┐
│   Profile   │ ← 人口统计、偏好、历史
│  Generator  │
└──────┬──────┘
       │
       ↓
┌─────────────┐
│   Memory    │ ← 短期记忆 + 长期记忆
│   Module    │
└──────┬──────┘
       │
       ↓
┌─────────────┐
│   Action    │ ← 浏览、点击、购买
│   Module    │
└─────────────┘
```

**Profile生成**：
- 人口统计（年龄、性别、位置）
- 偏好（类别、品牌、价格敏感度）
- 行为模式（活跃时间、频率）

**Memory模块**：
- **短期记忆**：最近浏览、当前会话
- **长期记忆**：历史偏好、购买记录

**Action模块**：
- 浏览行为
- 点击行为
- 购买行为

**实验结果**：

| 指标 | SimUSER | Random | Best Baseline |
|------|---------|--------|---------------|
| **行为预测准确率** | **85%** | 50% | 78% |
| **序列相似度** | **0.72** | 0.35 | 0.65 |
| **分布匹配度** | **0.89** | 0.42 | 0.81 |

**关键发现** ⭐⭐⭐：
- ✅ SimUSER在推荐场景**显著优于基线**
- ✅ Memory模块**关键**（移除后-15%）
- ✅ Profile质量**重要**（准确的人口统计+15%）
- ⚠️ **领域依赖**：不同推荐场景差异大

**相关性**：高（8/10 P1）

**核心价值**：
