## 📋 基本信息

**研究主题**: 评估AI代理在社会科学实验中的参与者模拟能力

**核心问题**: 
- LLM代理能否可靠模拟人类实验参与者？
- 代理设计如何影响模拟保真度？
- 如何在科学推断层面评估对齐度？

**核心贡献**:
1. **HumanStudy-Bench基准**：12个经典人类实验
2. **Filter-Extract-Execute-Evaluate流水线**：完整重建实验
3. **概率对齐分数（PAS）**：科学推断层面评估

---

## 🎯 核心方法

### 1. 四阶段流水线

```
Filter → Extract → Execute → Evaluate
(筛选)  (提取)    (执行)     (评估)
```

**Filter**：从文献中筛选人类实验
**Extract**：提取实验设计、参与者、统计检验
**Execute**：LLM代理执行实验
**Evaluate**：对比人类和代理结果

### 2. 12个经典实验

**个体认知**：
- 决策制定、判断任务

**策略交互**：
- 博弈论、经济学实验

**社会心理学**：
- 态度、偏见、群体行为

**规模**：
- 6,000+试验
- 参与者：数十到2,100+

### 3. 评估指标

**Probability Alignment Score (PAS)**：
```
PAS = π_human · π_agent + (1 - π_human) · (1 - π_agent)
```
- 评估科学推断层面的对齐
- 考虑人类基线的不确定性

**Effect Consistency Score (ECS)**：
- 评估效应量的一致性
- 数据层面的对齐

### 4. 代理设计

**四种设计**：
1. **Blank**：无角色
2. **Role-play**：角色扮演
3. **Demographic**：人口统计约束
4. **Rich backstory**：丰富背景故事

## 📊 核心发现

### 1. 有限且不一致的对齐

**PAS分数**：
- 当前LLM代理：**PAS < 0.6**（有限对齐）
- 人类 vs 代理：差异大

**行为模式**：
- LLM：**双峰分布**（极端化）
- 人类：**单峰分布**（温和）

### 2. 代理设计影响大

**非单调效应**：
- 更复杂的代理设计 ≠ 更好表现
- 有时简单设计更好

**设计敏感性**：
- 提示词、采样设置影响大
- 稳定性差

### 3. 领域依赖性强

**不同领域表现差异大**：
- 某些领域：LLM表现好
- 其他领域：LLM表现差
- 无通用最佳代理设计

### 4. 规模和集成不保证改进

**大模型**：
- 不一定更好
- 有时小模型表现更好

**多模型集成**：
- 不可靠提升
- 效果不一致

## 🎯 相关性评分

**评分**: **9/10 (P0)** ⭐⭐⭐⭐

**理由**:
- ✅ **完整实验重建**：12个经典实验
- ✅ **科学推断评估**：PAS + ECS
- ⚠️ **警告作用**：LLM代理对齐有限且不稳定

**关键价值**:
1. ✅ **基准测试**：可评估虚拟用户系统
2. ⚠️ **警告**：LLM模拟不可靠
3. ⚠️ **设计敏感**：代理设计影响大

---

## 📝 核心启示

**对虚拟用户研究的影响**（⚠️ 严重警告）:
1. **对齐有限**：PAS < 0.6，不可靠
2. **行为极端化**：双峰 vs 人类单峰
3. **设计敏感**：提示词、采样影响大
4. **领域依赖**：无通用最佳设计

**必须采取的措施**:
1. ✅ **使用PAS评估**：科学推断层面
2. ⚠️ **谨慎使用LLM代理**：对齐有限
3. ✅ **多代理设计测试**：寻找最佳设计
4. ✅ **真实数据验证**：对照P11+P12+P14警告
