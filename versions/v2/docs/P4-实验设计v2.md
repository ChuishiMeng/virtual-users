## 一、实验设计哲学

### 1.1 KDD方法论核心

```
传统实验设计: 单一数据集 → 少量baseline → 简单指标
         ↓
KDD实验设计: 多领域数据集 → 全面baseline → 四维评估 → 消融验证 → 泛化测试
```

### 1.2 实验目标分层

| 目标层级 | 具体目标 | 验证方法 |
|---------|---------|---------|
| **L1: 有效性** | 系统能否模拟真实用户响应？ | 主实验（分布对齐） |
| **L2: 优越性** | 是否显著优于现有方法？ | Baseline对比 |
| **L3: 可靠性** | 各模块是否必要？ | 系统性消融 |
| **L4: 泛化性** | 能否跨领域、跨任务应用？ | 跨域实验 |
| **L5: 可解释性** | 系统如何工作？ | 可视化+定性分析 |

### 2.3 评估指标四维框架

#### 维度1: 分布相似度（Distribution Alignment）

**主指标**:
```
KL散度 (Kullback-Leibler Divergence)
KL(P || Q) = Σ P(x) * log(P(x) / Q(x))
目标: KL < 0.5
```

**辅助指标**:
- **Jensen-Shannon散度**: KL的对称版本
- **Wasserstein距离**: 考虑分布间距离
- **Total Variation距离**: 最大差异

**粒度报告**:
- 整体KL（所有问题平均）
- 领域KL（按领域分组）
- 人口学KL（按年龄、性别、教育分组）

#### 维度2: 分析有效性（Analytical Validity）

| 指标 | 定义 | 目标 |
|-----|------|------|
| **子群差异显著性** | 虚拟数据能否发现与真实数据相同的子群差异 | p < 0.05一致性 > 80% |
| **效应量相关性** | 虚拟数据的效应量与真实数据的相关性 | r > 0.85 |
| **回归系数一致性** | 虚拟数据的回归系数与真实数据的一致性 | β相关性 > 0.80 |

#### 维度3: 行为一致性（Behavioral Consistency）

**内部一致性**:
```
Cohen's Kappa (跨问题一致性)
κ = (p_o - p_e) / (1 - p_e)
目标: κ > 0.7
```

**Test-Retest可靠性**:
- 间隔7天重复生成
- 计算响应相关性
- 目标: r > 0.8

**Persona-Response一致性**:
- 虚拟用户行为与Persona特征的逻辑一致性
- 人工评估（1-5分）
- 目标: > 4.0

#### 维度4: 交互真实性（Interaction Authenticity）

**图灵测试**:
```
Setup:
- 5位专家（市场调研×2，社会学家×2，AI研究者×1）
- 50个混合响应（25真实 + 25虚拟）
- 盲测区分

指标:
- 识别准确率（目标: < 60%，越低越好）
- 真实性评分（1-5分，目标: > 3.8）
```

**定性评估**:
- 响应深度（1-5分）
- 语言自然度（1-5分）
- 逻辑连贯性（1-5分）

## 四、泛化性验证（Generalization Test）

### 4.1 跨领域泛化（Cross-Domain）

**实验设计**: Leave-One-Domain-Out (LODO)

```
训练领域: {政治, 消费, 健康, 教育}
测试领域: 轮流留出一个

Setup:
- Train on 3 domains
- Test on held-out domain
- Repeat for all 4 domains
```

**预期结果**:
- 领域内KL: 0.42
- 跨领域KL: 0.48（+14%）
- 退化可接受
