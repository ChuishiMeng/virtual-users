## 一、统一表示空间 (Unified Representation Space)

### 1.1 问题定义

**核心挑战**: 不同问卷的题干、选项、量表各不相同，如何在统一空间中表示？

```
传统方法的问题：
Survey A: "您对该产品的满意度如何？" (1-5量表)
Survey B: "Would you recommend this product?" (NPS 0-10)
Survey C: "这个产品怎么样？" (开放式)

→ 表达异构 → Persona难以跨问卷迁移 → 难以复用知识
```

### 1.3 跨问卷迁移机制

```python
class CrossSurveyTransfer:
    """
    利用统一表示空间实现跨问卷知识迁移
    """
    
    def __init__(self):
        self.embed_cache = {}  # 问题嵌入缓存
        self.response_bank = {}  # 历史响应库
    
    def find_similar_questions(self, new_question, k=5):
        """
        在历史问卷中找到语义相似的问题
        """
        new_embed = unified_embed(new_question)
        similarities = {}
        
        for qid, cached_embed in self.embed_cache.items():
            sim = cosine_similarity(new_embed, cached_embed)
            similarities[qid] = sim
        
        # 返回Top-k相似问题
        return sorted(similarities, key=similarities.get, reverse=True)[:k]
    
    def transfer_persona_responses(self, persona_id, similar_questions):
        """
        将历史响应迁移到新问题
        """
        historical = self.response_bank[persona_id]
        transferred = {}
        
        for new_q, similar_q in similar_questions.items():
            # 相似问题 + 历史答案 → 新答案
            old_answer = historical[similar_q]
            transferred[new_q] = self.adapt_answer(old_answer, new_q)
        
        return transferred
```

## 二、Persona-RAG协同增强 (Synergistic Enhancement)

### 2.1 问题定义

**v1.0问题**: Persona和Retrieval是串行关系
```
Persona → Context → Retrieval → Response
         ↑ 弱连接，知识注入效率低
```

**v2.0目标**: 建立双向协同机制
```
Persona ⟷ Retrieval (联合优化)
    ↓
Enhanced Context
```

### 2.2 联合编码架构

#### 2.2.1 Persona-Aware Retrieval

```python
class PersonaAwareRetriever:
    """
    Persona指导检索的注意力机制
    """
    
    def __init__(self):
        self.doc_encoder = BERT()
        self.persona_encoder = PersonaBERT()
        self.cross_attention = CrossAttention()
    
    def retrieve(self, query, persona, knowledge_base, k=10):
        # Step 1: Persona条件化查询
        persona_query = self.persona_condition(query, persona)
        
        # Step 2: 检索候选文档
        candidates = self.initial_retrieval(persona_query, knowledge_base, k*2)
        
        # Step 3: Persona感知重排序
        reranked = self.persona_rerank(candidates, persona, query)
        
        return reranked[:k]
    
    def persona_condition(self, query, persona):
        """
        将Persona属性注入查询
        """
        # Persona特征: demographics + psychographics + tendencies
        persona_vec = self.persona_encoder(persona)
        query_vec = self.doc_encoder(query)
        
        # 条件化
        conditioned = self.cross_attention(
            query_vec, 
            persona_vec
        )
        return conditioned
    
    def persona_rerank(self, candidates, persona, query):
        """
        根据Persona偏好重排序
        """
        scores = []
        for doc in candidates:
            # 多因素评分
            relevance = self.relevance_score(query, doc)
            persona_match = self.persona_match_score(persona, doc)
            diversity = self.diversity_score(doc, candidates)
            
            # 加权融合
            score = 0.5*relevance + 0.3*persona_match + 0.2*diversity
            scores.append((doc, score))
        
        return sorted(scores, key=lambda x: x[1], reverse=True)
```

#### 2.2.2 Retrieval-Guided Persona Enhancement

```python
class RetrievalGuidedPersonaEnhancer:
    """
    检索结果反向增强Persona的领域知识
    """
    
    def enhance(self, persona, retrieved_docs):
        """
        根据检索到的知识，动态扩展Persona
        """
        # 提取领域知识
        domain_knowledge = self.extract_domain_knowledge(retrieved_docs)
        
        # 更新Persona的behavioral_context
        enhanced_persona = persona.copy()
        enhanced_persona.behavioral_context.update({
            "domain_expertise": domain_knowledge.expertise,
            "recent_experiences": domain_knowledge.examples,
            "pain_points": domain_knowledge.pain_points
        })
        
        return enhanced_persona
    
    def extract_domain_knowledge(self, docs):
        """
        从文档中提取结构化知识
        """
        knowledge = {
            "expertise": [],      # 领域专业术语
            "examples": [],       # 典型案例
            "pain_points": [],    # 常见痛点
            "trends": []          # 行业趋势
        }
        
        for doc in docs:
            # 使用LLM提取结构化信息
            extracted = self.llm_extract(doc)
            for key in knowledge:
                knowledge[key].extend(extracted.get(key, []))
        
        return knowledge
```

#### 3.1.2 自适应路由系统

```python
class AdaptiveResponseRouter:
    """
    根据问题类型自动选择最优生成策略
    """
    
    def __init__(self):
        self.generators = {
            "single_choice": CategoricalSampler(),
            "multiple_choice": MultiLabelClassifier(),
            "likert_scale": OrdinalRegressor(),
            "nps_scale": ContinuousRegressor(),
            "ranking": PreferenceRanker(),
            "open_ended": TextGenerator(),
            "matrix": MatrixCompleter()
        }
    
    def route_and_generate(self, question, persona, knowledge, type_probs):
        """
        混合策略生成
        """
        responses = {}
        
        for q_type, prob in type_probs.items():
            if prob > 0.1:  # 阈值
                generator = self.generators[q_type]
                responses[q_type] = generator.generate(
                    question, persona, knowledge
                )
        
        # 加权融合 (如果有多种类型)
        final_response = self.blend_responses(responses, type_probs)
        return final_response
```

```python
class PersonaConsistencyAdaptor:
    """
    确保生成响应与Persona属性一致
    """
    
    def __init__(self):
        self.consistency_checker = ConsistencyChecker()
    
    def adapt_response(self, response, persona, question):
        """
        检查并调整响应的Persona一致性
        """
        # 一致性检查
        consistency_score = self.consistency_checker.check(
            response, persona, question
        )
        
        if consistency_score < 0.7:
            # 一致性不足，重新生成
            return self.regenerate_with_constraints(
                persona, question, 
                constraints=self.extract_constraints(response)
            )
        
        return response
    
    def extract_constraints(self, failed_response):
        """
        从失败响应中提取约束
        """
        return {
            "avoid_patterns": [failed_response],
            "persona_alignment": "strict"
        }
```

## 四、迭代优化机制 (Iterative Optimization)

### 4.1 多轮对话反馈闭环

```
┌─────────────────────────────────────────────────────────┐
│           Multi-Turn Iterative Optimization             │
└─────────────────────────────────────────────────────────┘

Round 1:
  Question → Response₁ → Quality Score: 0.65
                    ↓
                Feedback: "响应过于泛泛，缺少具体细节"
                    ↓
Round 2:
  Question + Feedback → Response₂ → Quality Score: 0.82
                    ↓
                Feedback: "更好，但与Persona的risk_preference不符"
                    ↓
Round 3:
  Question + History → Response₃ → Quality Score: 0.91 ✅
```

### 4.2 迭代优化系统

```python
class IterativeOptimizer:
    """
    多轮迭代优化响应质量
    """
    
    def __init__(self, max_iterations=3, quality_threshold=0.85):
        self.max_iterations = max_iterations
        self.quality_threshold = quality_threshold
        self.quality_assessor = QualityAssessor()
        self.feedback_generator = FeedbackGenerator()
    
    def optimize(self, question, persona, initial_response):
        """
        迭代优化直到达到质量阈值
        """
        response = initial_response
        history = []
        
        for iteration in range(self.max_iterations):
            # 评估质量
            quality = self.quality_assessor.assess(
                response, persona, question
            )
            
            history.append({
                "iteration": iteration,
                "response": response,
                "quality": quality
            })
            
            # 达到阈值，停止
            if quality["overall"] >= self.quality_threshold:
                break
            
            # 生成反馈
            feedback = self.feedback_generator.generate(
                response, quality, persona
            )
            
            # 基于反馈重新生成
            response = self.regenerate(
                question, persona, feedback, history
            )
        
        return {
            "final_response": response,
            "iterations": len(history),
            "quality_history": history
        }
    
    def regenerate(self, question, persona, feedback, history):
        """
        基于反馈和历史重新生成
        """
        prompt = f"""
        Previous attempts:
        {self.format_history(history)}
        
        Feedback to address:
        {feedback}
        
        Persona:
        {persona}
        
        Question:
        {question}
        
        Generate an improved response:
        """
        
        return llm_generate(prompt)
```

### 4.3 质量评估器

```python
class QualityAssessor:
    """
    多维度质量评估
    """
    
    DIMENSIONS = {
        "relevance": {
            "weight": 0.3,
            "evaluator": "semantic_similarity"
        },
        "consistency": {
            "weight": 0.3,
            "evaluator": "persona_alignment"
        },
        "completeness": {
            "weight": 0.2,
            "evaluator": "coverage_check"
        },
        "naturalness": {
            "weight": 0.2,
            "evaluator": "fluency_score"
        }
    }
    
    def assess(self, response, persona, question):
        """
        多维度评估
        """
        scores = {}
        
        for dim, config in self.DIMENSIONS.items():
            evaluator = getattr(self, config["evaluator"])
            scores[dim] = evaluator(response, persona, question)
        
        # 加权综合
        overall = sum(
            scores[dim] * config["weight"]
            for dim, config in self.DIMENSIONS.items()
        )
        
        return {
            "overall": overall,
            "dimensions": scores,
            "weaknesses": self.identify_weaknesses(scores)
        }
```

### 4.4 反馈生成器

```python
class FeedbackGenerator:
    """
    生成可操作的改进反馈
    """
    
    def generate(self, response, quality, persona):
        """
        根据质量评估生成具体反馈
        """
        feedback_points = []
        
        for weakness in quality["weaknesses"]:
            if weakness == "low_relevance":
                feedback_points.append(
                    f"Response should be more specific to the question topic"
                )
            
            elif weakness == "low_consistency":
                mismatch = self.find_persona_mismatch(response, persona)
                feedback_points.append(
                    f"Response conflicts with persona attribute: {mismatch}"
                )
            
            elif weakness == "low_completeness":
                missing = self.find_missing_aspects(response, question)
                feedback_points.append(
                    f"Consider addressing: {missing}"
                )
            
            elif weakness == "low_naturalness":
                feedback_points.append(
                    f"Response sounds too formal/robotic, make it more natural"
                )
        
        return "\n".join(feedback_points)
```

```python
class UnifiedEmbeddingTrainer:
    """
    训练统一表示空间的编码器
    """
    
    def __init__(self):
        self.encoder = UnifiedEncoder()
        self.contrastive_loss = ContrastiveLoss()
    
    def train(self, survey_pairs):
        """
        对比学习训练
        survey_pairs: [(question1, question2, similarity_label), ...]
        """
        for q1, q2, label in survey_pairs:
            # 编码
            e1 = self.encoder(q1)
            e2 = self.encoder(q2)
            
            # 对比损失
            loss = self.contrastive_loss(e1, e2, label)
            
            # 反向传播
            self.optimizer.step(loss)
```

### 6.2 协同注意力实现

```python
class SynergisticCrossAttention(nn.Module):
    """
    Persona-Knowledge协同注意力
    """
    
    def __init__(self, hidden_size=768, num_heads=8):
        super().__init__()
        self.attention = nn.MultiheadAttention(hidden_size, num_heads)
        self.persona_proj = nn.Linear(hidden_size, hidden_size)
        self.knowledge_proj = nn.Linear(hidden_size, hidden_size)
    
    def forward(self, persona_emb, knowledge_emb):
        # 投影
        P = self.persona_proj(persona_emb)
        K = self.knowledge_proj(knowledge_emb)
        
        # 双向注意力
        # Persona attends to Knowledge
        P_enhanced, _ = self.attention(P, K, K)
        
        # Knowledge attends to Persona
        K_enhanced, _ = self.attention(K, P, P)
        
        # 残差连接
        synergistic = P + P_enhanced + K_enhanced
        
        return synergistic
```

### 6.3 自适应生成器

```python
class AdaptiveResponseGenerator:
    """
    根据问题类型自适应生成
    """
    
    def __init__(self):
        self.type_classifier = QuestionTypeClassifier()
        self.generators = {
            "discrete": DiscreteGenerator(),
            "ordinal": OrdinalGenerator(),
            "continuous": ContinuousGenerator(),
            "open": OpenGenerator()
        }
    
    def generate(self, question, persona, knowledge):
        # 分类
        type_probs = self.type_classifier.classify(question)
        
        # 混合生成
        responses = []
        for type_name, prob in type_probs.items():
            if prob > 0.1:
                gen = self.generators[type_name]
                response = gen.generate(question, persona, knowledge)
                responses.append((response, prob))
        
        # 加权选择
        return self.weighted_select(responses)
```
