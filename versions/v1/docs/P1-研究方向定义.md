## 🎯 研究主题

**用大模型构造虚拟用户替代传统人工问卷调研**

### 问题背景

传统用户调研依赖人工问卷：
- 成本高（招募、激励、时间）
- 样本偏差（谁愿意填问卷？）
- 周期长（设计→发放→回收→分析）
- 难以大规模、高频次调研

### 研究方向

利用大语言模型（LLM）生成**虚拟用户**（Virtual/Synthetic Users），模拟真实用户的行为、偏好、决策，完成自动化用户调研。
