## ✅ 滚雪球检索方法总结

**使用工具**：
- **Semantic Scholar API** - 获取论文引用和参考文献
- API 端点：`https://api.semanticscholar.org/graph/v1/paper/arXiv:{ID}?fields=references,citations`

**筛选标准**：
1. 引用数 > 20（表示影响力）
2. 标题包含关键词：survey, simulation, persona, LLM, human, respondent
3. 与研究方向相关性高

**局限性**：
- Semantic Scholar 有速率限制（429）
- 新论文（如 Polypersona）尚未被引用
