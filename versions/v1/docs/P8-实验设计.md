# P8: 虚拟用户研究 - 实验设计

## 研究主题
**用大模型构造虚拟用户替代传统人工问卷调研**

---

## 1. 实验目标

### 1.1 核心研究问题

**RQ1: 虚拟用户系统能否生成与真实用户一致的调研响应？**
- 子问题 1.1：单题层面的准确性如何？
- 子问题 1.2：聚合层面的分布一致性如何？
- 子问题 1.3：不同问题类型的准确性差异？

**RQ2: 各模块（Persona、检索、响应）对最终效果的贡献如何？**
- 子问题 2.1：Persona 的必要性？
- 子问题 2.2：检索增强的价值？
- 子问题 2.3：系统设计的合理性？

**RQ3: 虚拟用户系统在不同场景下的泛化能力如何？**
- 子问题 3.1：跨领域的表现？
- 子问题 3.2：跨人群的表现？
- 子问题 3.3：跨问卷类型的表现？

### 1.2 实验假设

| 假设 | 描述 | 验证方法 |
|------|------|----------|
| **H1** | 虚拟用户响应与真实用户响应的分布相似度 > 0.8 | 分布检验 |
| **H2** | Persona 生成能显著提升响应真实性 | 消融实验 |
| **H3** | 检索增强能提升响应准确性和一致性 | 消融实验 |
| **H4** | 系统在消费电子领域表现优于其他领域 | 跨领域实验 |

### 3.2 评估指标

#### 3.2.1 单题层面指标

```python
def compute_single_question_metrics(real_answers, virtual_answers, question_type):
    """
    计算单题层面指标
    """
    metrics = {}
    
    if question_type == "single_choice":
        # 准确率（最常见选项匹配）
        real_mode = mode(real_answers)
        virtual_mode = mode(virtual_answers)
        metrics['mode_match'] = 1 if real_mode == virtual_mode else 0
        
        # Top-3 重叠度
        real_top3 = get_top_k(real_answers, k=3)
        virtual_top3 = get_top_k(virtual_answers, k=3)
        metrics['top3_overlap'] = len(set(real_top3) & set(virtual_top3)) / 3
    
    elif question_type == "multiple_choice":
        # Jaccard 相似度
        real_set = set(real_answers)
        virtual_set = set(virtual_answers)
        metrics['jaccard'] = len(real_set & virtual_set) / len(real_set | virtual_set)
    
    elif question_type == "likert_scale":
        # 均值误差
        real_mean = np.mean(real_answers)
        virtual_mean = np.mean(virtual_answers)
        metrics['mean_error'] = abs(real_mean - virtual_mean)
        
        # 分布差异（KL 散度）
        metrics['kl_divergence'] = compute_kl_divergence(real_answers, virtual_answers)
    
    elif question_type == "open_ended":
        # 语义相似度（BERTScore）
        metrics['semantic_similarity'] = compute_bertscore(real_answers, virtual_answers)
        
        # 主题一致性（LDA）
        real_topics = extract_topics(real_answers)
        virtual_topics = extract_topics(virtual_answers)
        metrics['topic_nmi'] = compute_nmi(real_topics, virtual_topics)
    
    return metrics
```

#### 3.2.2 聚合层面指标

```python
def compute_aggregate_metrics(real_responses, virtual_responses):
    """
    计算聚合层面指标
    """
    metrics = {}
    
    # 1. 整体分布相似度
    metrics['distribution_similarity'] = compute_overall_distribution_similarity(
        real_responses, virtual_responses
    )
    
    # 2. 关键指标误差
    # 假设关注品牌偏好、价格敏感度等关键指标
    key_indicators = ['brand_preference', 'price_sensitivity', 'feature_importance']
    
    for indicator in key_indicators:
        real_value = compute_indicator(real_responses, indicator)
        virtual_value = compute_indicator(virtual_responses, indicator)
        
        metrics[f'{indicator}_error'] = abs(real_value - virtual_value) / real_value
    
    # 3. 细分人群一致性
    segments = ['age_group', 'gender', 'city_tier']
    
    for segment in segments:
        metrics[f'{segment}_consistency'] = compute_segment_consistency(
            real_responses, virtual_responses, segment
        )
    
    return metrics
```

#### 3.2.3 统计显著性检验

```python
def statistical_tests(real_responses, virtual_responses, question_id):
    """
    统计显著性检验
    """
    real_answers = extract_answers(real_responses, question_id)
    virtual_answers = extract_answers(virtual_responses, question_id)
    
    results = {}
    
    # 卡方检验（分类变量）
    if is_categorical(real_answers):
        contingency_table = build_contingency_table(real_answers, virtual_answers)
        chi2, p_value = chi2_test(contingency_table)
        results['chi_square'] = {'statistic': chi2, 'p_value': p_value}
    
    # t 检验（连续变量）
    if is_continuous(real_answers):
        t_stat, p_value = ttest_ind(real_answers, virtual_answers)
        results['t_test'] = {'statistic': t_stat, 'p_value': p_value}
        
        # 效应量
        results['cohens_d'] = compute_cohens_d(real_answers, virtual_answers)
    
    # KS 检验（分布差异）
    ks_stat, p_value = ks_test(real_answers, virtual_answers)
    results['ks_test'] = {'statistic': ks_stat, 'p_value': p_value}
    
    return results
```

```python
def ablation_study(survey, real_responses, configs):
    """
    消融实验
    """
    results = {}
    
    for config_name, config in configs.items():
        # 根据配置生成虚拟响应
        virtual_responses = generate_virtual_responses(
            survey,
            use_persona=config['persona'],
            use_retrieval=config['retrieval']
        )
        
        # 计算指标
        metrics = compute_all_metrics(real_responses, virtual_responses)
        results[config_name] = metrics
    
    # 计算贡献度
    contribution = {
        'persona_contribution': results['Full']['accuracy'] - results['w/o Persona']['accuracy'],
        'retrieval_contribution': results['Full']['accuracy'] - results['w/o Retrieval']['accuracy'],
        'both_contribution': results['Full']['accuracy'] - results['w/o Both']['accuracy']
    }
    
    return results, contribution
```

## 10. 伦理与安全考虑

### 10.1 伦理风险

| 风险 | 描述 | 缓解措施 |
|------|------|---------|
| **误导性结论** | 虚拟响应可能误导决策 | 明确标注数据来源，提供置信度 |
| **偏见放大** | LLM 偏见传递到虚拟响应 | 多模型对比，偏见检测 |
| **滥用风险** | 用于操纵调研结果 | 使用协议，审计追踪 |

### 10.2 安全措施

**数据安全**：
- 真实数据脱敏处理
- 访问权限控制
- 数据加密存储

**系统安全**：
- API 调用频率限制
- 输入验证和过滤
- 异常检测和告警
