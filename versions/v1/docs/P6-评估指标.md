# P6: 评估指标体系

本文档整理虚拟用户生成系统的评估指标，涵盖分布对齐、一致性、准确性和多样性四个维度。

---

## 1. 分布对齐指标 (Distribution Alignment Metrics)

### 1.1 KL Divergence (Kullback-Leibler 散度)

#### 定义/公式
```
D_KL(P || Q) = Σ P(x) * log(P(x) / Q(x))
```
对于连续分布：
```
D_KL(P || Q) = ∫ P(x) * log(P(x) / Q(x)) dx
```

#### 适用场景
- 衡量两个概率分布之间的差异
- 评估生成分布与真实分布的偏离程度
- 信息论和机器学习中的分布匹配

#### 取值范围
- **范围**: [0, +∞)
- **解释**: 0 表示两个分布完全相同，值越大差异越大
- **非对称性**: D_KL(P || Q) ≠ D_KL(Q || P)

#### 使用示例
```python
import numpy as np
from scipy.stats import entropy

# 真实分布和生成分布
real_dist = np.array([0.2, 0.3, 0.3, 0.2])
generated_dist = np.array([0.25, 0.25, 0.25, 0.25])

# 计算 KL 散度
kl_div = entropy(real_dist, generated_dist)  # ~0.038
```

---

### 1.2 JS Distance (Jensen-Shannon 距离)

#### 定义/公式
```
JS(P || Q) = (1/2) * D_KL(P || M) + (1/2) * D_KL(Q || M)
```
其中 M = (P + Q) / 2

JS Distance = √JS(P || Q)

#### 适用场景
- 需要对称性度量时（对比 KL 散度）
- 生成模型评估（如 GAN）
- 文本和用户画像分布比较

#### 取值范围
- **JS 散度**: [0, log(2)] ≈ [0, 0.693]
- **JS 距离**: [0, 1]（归一化后）
- **对称性**: JS(P || Q) = JS(Q || P)

#### 使用示例
```python
from scipy.spatial.distance import jensenshannon

real_dist = np.array([0.2, 0.3, 0.3, 0.2])
generated_dist = np.array([0.25, 0.25, 0.25, 0.25])

# JS 距离
js_dist = jensenshannon(real_dist, generated_dist)  # ~0.135
```

### 1.3 Wasserstein Distance (推土机距离/EMD)

#### 定义/公式
对于一维分布：
```
W_p(P, Q) = (∫ |F_P(x) - F_Q(x)|^p dx)^(1/p)
```
其中 F_P 和 F_Q 是累积分布函数

对于离散分布（1-Wasserstein）：
```
W_1(P, Q) = Σ |CDF_P(i) - CDF_Q(i)| * Δx
```

#### 适用场景
- GAN 训练（Wasserstein GAN）
- 分布差异较大时仍能提供有意义的梯度
- 年龄、收入等连续属性的分布对齐

#### 取值范围
- **范围**: [0, +∞)
- **解释**: 0 表示分布完全相同
- **优势**: 即使分布不重叠也能提供有意义的度量

#### 使用示例
```python
from scipy.stats import wasserstein_distance

# 年龄分布
real_ages = [25, 30, 35, 40, 45, 50, 55, 60]
real_weights = [0.1, 0.15, 0.2, 0.2, 0.15, 0.1, 0.05, 0.05]

generated_ages = [25, 30, 35, 40, 45, 50, 55, 60]
generated_weights = [0.08, 0.12, 0.18, 0.22, 0.18, 0.12, 0.06, 0.04]

w_dist = wasserstein_distance(real_ages, generated_ages, 
                               real_weights, generated_weights)
```

## 2. 一致性指标 (Consistency Metrics)

### 2.1 Self-consistency Score (自一致性分数)

#### 定义/公式
```
Consistency = (1/N) * Σ I(response_i^(t1) == response_i^(t2))
```
或对于连续值：
```
Consistency = 1 - (1/N) * Σ |response_i^(t1) - response_i^(t2)| / max_diff
```

#### 适用场景
- 同一虚拟用户对相同问题的回答一致性
- LLM 生成稳定性评估
- 虚拟用户行为可靠性验证

#### 取值范围
- **范围**: [0, 1]
- **解释**: 1 表示完全一致，0 表示完全不一致
- **建议阈值**: > 0.8 表示良好一致性

#### 使用示例
```python
import numpy as np

# 同一虚拟用户两次回答相同问卷
responses_t1 = ['A', 'B', 'A', 'C', 'B']
responses_t2 = ['A', 'B', 'A', 'A', 'B']

# 计算一致性
matches = sum(r1 == r2 for r1, r2 in zip(responses_t1, responses_t2))
consistency = matches / len(responses_t1)  # 0.8
```

---

### 2.2 Test-Retest Reliability (重测信度)

#### 定义/公式
Pearson 相关系数：
```
r = Σ(x_i - x̄)(y_i - ȳ) / √[Σ(x_i - x̄)² * Σ(y_i - ȳ)²]
```
或使用 Cronbach's Alpha：
```
α = (K / (K-1)) * (1 - Σσ_i² / σ_t²)
```

#### 适用场景
- 问卷/量表稳定性评估
- 虚拟用户画像一致性验证
- 心理测量学指标迁移

#### 取值范围
- **Pearson r**: [-1, 1]
- **Cronbach's α**: [0, 1]
- **解释**: 
  - r > 0.7: 良好信度
  - α > 0.8: 优秀信度

#### 使用示例
```python
from scipy.stats import pearsonr

# 两次测试的分数
scores_t1 = [3.5, 4.2, 2.8, 4.0, 3.9]
scores_t2 = [3.6, 4.0, 2.9, 4.1, 3.8]

# 计算重测信度
reliability, p_value = pearsonr(scores_t1, scores_t2)
# reliability ≈ 0.98, p < 0.01
```

## 3. 准确性指标 (Accuracy Metrics)

### 3.1 Accuracy (准确率)

#### 定义/公式
```
Accuracy = (TP + TN) / (TP + TN + FP + FN)
```
多分类：
```
Accuracy = (1/N) * Σ I(ŷ_i == y_i)
```

#### 适用场景
- 虚拟用户行为预测
- 用户分类任务
- 平衡数据集的评估

#### 取值范围
- **范围**: [0, 1]
- **解释**: 1 表示完全正确
- **注意**: 类别不平衡时可能误导

#### 使用示例
```python
from sklearn.metrics import accuracy_score

y_true = [0, 1, 1, 0, 1, 0, 1, 1]
y_pred = [0, 1, 0, 0, 1, 0, 1, 1]

acc = accuracy_score(y_true, y_pred)  # 0.875
```

---

### 3.2 F1 Score

#### 定义/公式
```
Precision = TP / (TP + FP)
Recall = TP / (TP + FN)
F1 = 2 * (Precision * Recall) / (Precision + Recall)
```
Macro-F1（多分类）：
```
F1_macro = (1/K) * Σ F1_k
```

#### 适用场景
- 类别不平衡的分类任务
- 需要平衡精确率和召回率
- 虚拟用户兴趣/行为预测

#### 取值范围
- **范围**: [0, 1]
- **解释**: 1 表示完美平衡
- **F1 > 0.7**: 通常认为良好

#### 使用示例
```python
from sklearn.metrics import f1_score

y_true = [0, 1, 1, 0, 1, 0, 1, 1]
y_pred = [0, 1, 0, 0, 1, 0, 1, 1]

f1 = f1_score(y_true, y_pred, average='binary')  # 0.857
f1_macro = f1_score(y_true, y_pred, average='macro')
```

---

### 3.3 AUC (Area Under ROC Curve)

#### 定义/公式
```
AUC = P(score(positive) > score(negative))
```
即随机正样本得分高于随机负样本得分的概率

#### 适用场景
- 二分类任务概率预测
- 排序质量评估
- 阈值无关的模型比较

#### 取值范围
- **范围**: [0, 1]
- **解释**:
  - AUC = 0.5: 随机猜测
  - AUC = 1.0: 完美分类
  - AUC > 0.8: 良好性能

#### 使用示例
```python
from sklearn.metrics import roc_auc_score

y_true = [0, 0, 1, 1, 0, 1, 0, 1]
y_prob = [0.1, 0.3, 0.8, 0.9, 0.2, 0.7, 0.4, 0.85]

auc = roc_auc_score(y_true, y_prob)  # ~0.94
```

## 4. 多样性指标 (Diversity Metrics)

### 4.1 Entropy (信息熵)

#### 定义/公式
Shannon 熵：
```
H(X) = -Σ p(x) * log₂ p(x)
```
归一化熵：
```
H_norm = H(X) / log₂(N)
```
其中 N 为类别数

#### 适用场景
- 虚拟用户属性分布多样性
- 生成内容的丰富度
- 用户行为模式的多样性

#### 取值范围
- **Shannon 熵**: [0, log₂(N)]
- **归一化熵**: [0, 1]
- **解释**: 值越大越多样，0 表示完全集中

#### 使用示例
```python
from scipy.stats import entropy
import numpy as np

# 用户兴趣分布
interests = np.array([0.3, 0.25, 0.2, 0.15, 0.1])

# 计算熵（以 2 为底）
H = entropy(interests, base=2)  # ~2.19 bits

# 归一化熵
H_norm = H / np.log2(len(interests))  # ~0.93
```

---

### 4.2 Distinct-n (去重率)

#### 定义/公式
```
Distinct-n = |unique n-grams| / |total n-grams|
```
常见变体：
- **Distinct-1**: 单词级别去重率
- **Distinct-2**: 双词组合去重率

#### 适用场景
- 文本生成多样性评估
- 虚拟用户评论/反馈生成
- 对话系统回复多样性

#### 取值范围
- **范围**: [0, 1]
- **解释**: 1 表示完全无重复，0 表示完全重复
- **建议**: Distinct-2 > 0.6 为良好

#### 使用示例
```python
def distinct_n(texts, n=2):
    """计算 Distinct-n 分数"""
    all_ngrams = []
    for text in texts:
        words = text.split()
        ngrams = [' '.join(words[i:i+n]) for i in range(len(words)-n+1)]
        all_ngrams.extend(ngrams)
    
    if len(all_ngrams) == 0:
        return 0.0
    
    return len(set(all_ngrams)) / len(all_ngrams)

generated_texts = [
    "I love this product very much",
    "This is a great amazing product",
    "Really love the quality here"
]

d2 = distinct_n(generated_texts, n=2)  # ~0.87
```

### 4.3 多样性评分 (Diversity Score)

#### 定义/公式
组合多样性评分：
```
Diversity_Score = w1 * Entropy + w2 * Distinct + w3 * Coverage
```
其中：
- **Coverage** = |unique items| / |vocabulary size|

余弦相似度多样性：
```
Diversity = 1 - (2/N(N-1)) * Σ cos_sim(v_i, v_j)
```

#### 适用场景
- 综合评估虚拟用户群体多样性
- 多维度多样性指标融合
- 生成系统整体质量评估

#### 取值范围
- **范围**: [0, 1]
- **解释**: 1 表示最大多样性
- **权重**: 根据任务需求调整

#### 使用示例
```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

def diversity_score(embeddings):
    """基于嵌入的多样性评分"""
    n = len(embeddings)
    if n < 2:
        return 1.0
    
    # 计算所有嵌入对的余弦相似度
    sim_matrix = cosine_similarity(embeddings)
    
    # 排除对角线（自相似度）
    np.fill_diagonal(sim_matrix, 0)
    
    # 平均相似度
    avg_sim = np.sum(sim_matrix) / (n * (n - 1))
    
    # 转换为多样性分数
    return 1 - avg_sim

# 示例：虚拟用户嵌入向量
user_embeddings = np.random.rand(100, 128)  # 100 个用户，128 维
div_score = diversity_score(user_embeddings)
```

## 5. 指标选择指南

### 按任务类型选择

| 任务类型 | 推荐指标 |
|---------|---------|
| 分布对齐 | Wasserstein > JS > KL（按鲁棒性排序） |
| 生成稳定性 | Self-consistency + Test-Retest |
| 分类预测 | F1 + AUC（不平衡数据首选） |
| 内容多样性 | Entropy + Distinct-2 |

### 按场景选择

| 场景 | 核心指标 | 辅助指标 |
|-----|---------|---------|
| 虚拟用户画像生成 | JS Distance | Entropy |
| 行为预测 | F1, AUC | Accuracy |
| 评论/反馈生成 | Distinct-2 | Entropy |
| 用户分布模拟 | Wasserstein | KL Divergence |
