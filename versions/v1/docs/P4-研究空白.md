## 🔍 详细研究空白（更新）

### 空白 1: 市场调研问卷场景缺失 ⭐⭐⭐⭐⭐

**现状**：
- AlignSurvey 覆盖社会调查流程，但**非商业市场调研**
- LLM-S³ 覆盖社会人口学调查，但**非消费者行为**
- Polypersona 针对问卷回答生成，但**非市场调研专用**

**机会**：
- 市场调研是巨大的商业需求（全球数百亿美元市场）
- 传统问卷成本高、周期长、样本偏差
- 虚拟用户可大幅降低成本、提高效率、扩大样本

**创新点**：
- 产品测试场景（概念测试、包装测试、定价测试）
- 品牌调研场景（品牌认知、品牌形象）
- 用户满意度场景（NPS、CSAT）

---

### 空白 2: 可靠性问题未完全解决 ⭐⭐⭐⭐⭐

**问题证据**（来自 Lost in Simulation）：
```
LLM 模拟用户的可靠性问题：
├── 跨LLM差异（9%）
├── 跨人群/方言差异（AAVE, 印度英语等）
├── 系统性校准错误（低估困难、高估中等）
└── 失败模式不一致
```

**现有方案局限**：

| 方案 | 论文 | 解决程度 |
|------|------|----------|
| 分布对齐 | Distribution Shift | 部分 |
| 不确定性量化 | Uncertainty Quantification | 评估方法 |
| 上下文增强 | LLM-S³ FAS | 部分 |
| 检索约束 | PersonaCite | 高（但依赖数据） |

**机会**：
- **组合方案**：分布对齐 + 检索约束 + 不确定性量化
- **自适应校准**：根据任务难度动态调整
- **多模型集成**：减少单一模型偏差

---

### 空白 3: 跨人群公平性 ⭐⭐⭐⭐

**问题**：
- AAVE（非裔英语）用户成功率更低
- 印度英语、肯尼亚英语表现差
- 年龄复合歧视效应

**现有研究**：
- AlignSurvey 关注公平性评估
- 但**未提出解决方案**

**机会**：
- 多方言/多人群训练数据
- 公平性约束优化
- 偏差检测与校正

---

### 空白 4: 问卷设计的特殊需求 ⭐⭐⭐⭐

**市场调研问卷特点**：

| 特点 | 挑战 | 现有研究 |
|------|------|----------|
| 开放式问题 | 生成多样化、真实回答 | Polypersona 部分 |
| 量表题 | 保持一致偏好模式 | ❌ |
| 逻辑跳转 | 根据回答动态调整 | ❌ |
| 多轮交互 | 长期一致性 | Generative Agents |
| 概念测试 | 理解新产品概念 | ❌ |
| 对比测试 | 多选项对比偏好 | ❌ |

**机会**：
- 问卷结构感知的响应生成
- 偏好一致性约束
- 概念理解增强

---

### 空白 5: 可验证性与来源追溯 ⭐⭐⭐⭐

**问题**：
- 大多数方法无来源约束
- 可能生成无根据的回答
- 难以验证响应真实性

**现有方案**：
- PersonaCite：VoC检索增强
- 但仅限有VoC数据的场景

**机会**：
- **多源检索**：VoC + 评论 + 社交媒体
- **响应级别归因**：每个回答可追溯
- **置信度标注**：低置信回答标记

## 🎯 推荐研究方向（最终版）

### 方向 A: 可靠虚拟用户问卷框架（推荐 ⭐⭐⭐⭐⭐）

**核心贡献**：
1. **首个针对市场调研问卷的虚拟用户框架**
2. **问卷结构感知**：处理开放式、量表、逻辑跳转
3. **组合可靠性增强**：分布对齐 + 检索约束 + 不确定性量化

**技术路线**：
```
输入：问卷 + 产品/服务描述
        ↓
┌─────────────────────────────────────┐
│ Step 1: Persona 生成                │
│ • 目标人群统计分布                   │
│ • 消费者背景、偏好、态度             │
│ • 基于真实消费者数据（如有的话）     │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ Step 2: 多源检索增强                 │
│ • 产品评论/反馈                      │
│ • 社交媒体讨论                       │
│ • 行业报告                           │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ Step 3: 问卷结构感知响应生成         │
│ • 理解问题类型（开放式/量表/选择）   │
│ • 保持偏好一致性                     │
│ • 处理逻辑跳转                       │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│ Step 4: 可靠性评估                   │
│ • 分布对齐检查                       │
│ • 不确定性量化                       │
│ • 置信度标注                         │
└─────────────────────────────────────┘
        ↓
输出：问卷回答 + 可靠性评分 + 来源归因
```

**评估方法**：
- 使用 LLM-S³ 和 AlignSurvey 作为评估基准
- 与 Polypersona 对比
- 真实问卷数据验证（如有）
