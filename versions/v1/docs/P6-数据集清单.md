## 3. LLM 评估基准数据集

### 3.1 LLM-S³ Benchmark ⭐ 核心基准

| 属性 | 信息 |
|------|------|
| **论文** | arXiv:2509.06337 |
| **规模** | 11 个真实世界公共数据集 |
| **领域** | 4 个社会学领域 |
| **访问方式** | https://github.com/dart-lab-research/LLM-S-Cube-Benchmark |
| **许可** | 开源 |
| **品牌相关性** | ⭐⭐⭐⭐⭐ **直接相关**，问卷模拟基准 |

**核心方法**：
- **PAS (Partial Attribute Simulation)**：基于部分 Profile 预测缺失属性
- **FAS (Full Attribute Simulation)**：零上下文 + 上下文增强生成完整数据集

**包含数据集**：需查看论文补充材料获取 11 个数据集详细列表

### 3.2 AlignSurvey Benchmark ⭐ 完整流程基准

| 属性 | 信息 |
|------|------|
| **论文** | arXiv:2511.07871 |
| **规模** | Social Foundation Corpus (44K+ 对话，400K+ 调查记录) |
| **访问方式** | 论文补充材料 |
| **许可** | 待确认 |
| **品牌相关性** | ⭐⭐⭐⭐⭐ **直接相关**，完整调查流程 |

**四阶段任务**：
1. Social Role Modeling (社会角色建模)
2. Semi-structured Interview Modeling (半结构化访谈)
3. Attitude Stance Modeling (态度立场建模)
4. Survey Response Modeling (调查响应建模)

**数据组成**：
- Social Foundation Corpus（预训练语料）
- AlignSurvey-Expert (ASE)（专家标注）
- 全国代表性调查（跨文化评估）

### 3.3 HumanStudy-Bench

| 属性 | 信息 |
|------|------|
| **论文** | arXiv:2602.00685 |
| **规模** | 12 项基础研究，6000+ 试验 |
| **品牌相关性** | ⭐⭐⭐⭐ 参与者模拟评估 |

**Agent 设计**：Agent = 基础模型 + 规格说明
**流程**：Filter → Extract → Execute → Evaluate
