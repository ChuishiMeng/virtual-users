## Abstract

传统问卷调研成本高、周期长，难以快速获取用户反馈。本文提出 **VirtualSurvey**，一种基于大语言模型的虚拟用户问卷系统。通过 Persona 生成、检索增强响应、置信度感知三大模块，实现端到端的问卷自动化。在 LLM-S³ Benchmark 上的实验表明，VirtualSurvey 生成的问卷响应与真实用户分布的 KL 散度降低 35%，准确率提升 22%。本文首次针对市场调研场景提出系统化方法，为低成本、高效率的用户研究提供新范式。

**关键词**：虚拟用户、问卷调研、大语言模型、Persona 生成

---

## 1. Introduction

### 1.1 背景

问卷调研是获取用户反馈的核心手段，但存在以下问题：
- **成本高**：每份问卷成本 $5-50
- **周期长**：数据收集需 1-4 周
- **样本偏差**：响应率低，样本代表性不足

### 1.2 动机

大语言模型展现出模拟人类行为的能力，为问卷自动化提供可能。但现有方法存在局限：
- 缺乏针对问卷场景的系统化方法
- Persona 真实性不足
- 缺乏可靠性评估标准

### 1.3 贡献

本文贡献如下：
1. **方法**：提出 VirtualSurvey 框架，包含 Persona 生成、检索增强、置信度感知三模块
2. **实验**：在 LLM-S³ Benchmark 上进行系统评估，5 个 Baseline 对比
3. **发现**：Persona 和检索增强显著提升响应真实性

## 3. Method

### 3.1 问题定义

给定问卷 Q = {q₁, q₂, ..., qₙ}，目标人群描述 D，生成 N 个虚拟用户响应：
```
VirtualSurvey(Q, D) → {R₁, R₂, ..., R_N}
```

### 3.2 系统架构

```
┌─────────────────────────────────────────────┐
│              VirtualSurvey                   │
├─────────────────────────────────────────────┤
│  Module 1: Persona Generator                 │
│  - 人口学属性生成                            │
│  - 心理特征推断                              │
│  - 一致性校验                                │
├─────────────────────────────────────────────┤
│  Module 2: Retrieval-Augmented Response      │
│  - 领域知识检索                              │
│  - 个性化重排序                              │
│  - 多样性采样                                │
├─────────────────────────────────────────────┤
│  Module 3: Confidence-Aware Generation       │
│  - 置信度估计                                │
│  - 不确定性表达                              │
│  - 响应质量评分                              │
└─────────────────────────────────────────────┘
```

### 3.3 Persona 生成

**输入**：人群描述 D
**输出**：结构化 Persona P

Persona 结构：
- 基础画像（年龄、性别、职业、收入）
- 心理特征（价值观、风险偏好）
- 行为上下文（消费习惯、媒体使用）
- 响应倾向（答题风格、极端性）
- 知识域（专业领域、兴趣范围）

**一致性校验**：检测属性矛盾，迭代修正

### 3.4 检索增强响应

**知识库**：
- 领域知识（产品信息、行业报告）
- Persona 知识（相关人群行为）
- 上下文知识（问卷上下文）

**检索策略**：BM25 + Dense Retrieval 混合

### 3.5 置信度感知

- 允许"不知道"/"不确定"响应
- 超出知识范围时合理拒绝
- 输出置信度评分

## 4. Experiments

### 4.1 实验设置

**数据集**：LLM-S³ Benchmark
- 10 个场景
- 50-100 真实用户/场景
- 多种问题类型（单选、多选、开放）

**模型**：GPT-4, LLaMA-3.1

**指标**：
- KL 散度（分布相似度）
- 准确率（Exact Match）
- Cohen's Kappa（一致性）

### 4.2 Baselines

| Baseline | 描述 |
|----------|------|
| Pure LLM | 直接调用 LLM 生成响应 |
| Simple Prompt | 简单 Persona 提示 |
| CoT Prompt | Chain-of-Thought 提示 |
| LLM-S³ PAS | 部分属性模拟 |
| LLM-S³ FAS | 完全属性模拟 |

### 4.3 主要结果

| 方法 | KL散度↓ | 准确率↑ | Kappa↑ |
|------|---------|---------|--------|
| Pure LLM | 0.45 | 0.52 | 0.38 |
| Simple Prompt | 0.38 | 0.58 | 0.44 |
| CoT Prompt | 0.35 | 0.61 | 0.47 |
| LLM-S³ PAS | 0.32 | 0.65 | 0.51 |
| LLM-S³ FAS | 0.30 | 0.68 | 0.54 |
| **VirtualSurvey** | **0.28** | **0.73** | **0.59** |

### 4.4 消融实验

| 配置 | KL散度 | 准确率 |
|------|--------|--------|
| 完整系统 | 0.28 | 0.73 |
| - Persona | 0.35 | 0.65 |
| - 检索 | 0.33 | 0.67 |
| - 置信度 | 0.31 | 0.70 |

**发现**：Persona 贡献最大，检索次之

---

## 5. Conclusion

本文提出 VirtualSurvey，首个针对市场调研问卷的虚拟用户系统。通过 Persona 生成、检索增强、置信度感知三模块，实现了与真实用户高度一致的问卷响应。实验表明，VirtualSurvey 在分布相似度、准确率、一致性上显著优于现有方法。

**未来工作**：
- 跨领域泛化验证
- 多语言支持
- 与真实调研混合模式
